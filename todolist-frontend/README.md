### 초기 설치
    - npx create-react-app todolist-frontend
    - npm i axios
    - npm i react-router-dom

## 개발 중점
    1. 페이지 (부모,자식) 컴포넌트 분리 (props 활용)
    2. 데이터 값 변경시 마다 SELECT 문 실행을 최소화 
         - 클라이언트 내부에서 (map, filter)등 활용하여 처리하여 DB조회 리소스 낭비 최소화
